# Colored Regions for Visual Studio Code

This package provides a simple way of colorizing regions.

## Features

Customize your regions by providing a `rgba(r, g, b, a)` color.

```javascript
//#region[rgba(255, 255, 255, 0.15)] Hello region
const hello = () => 'region';
//#endregion
```
